## 1. Laravel Project Installation Instructions
- first clone the repository
- git clone repository_link
- cd project_name
- cp .env.example .env
- composer install
- composer update
- php artisan key:generate
-  open .env and update DB_DATABASE (database details)
-  php artisan migrate:fresh --seed
- php artisan serve
